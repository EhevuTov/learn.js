Sometimes, you might want to have your program behave differently depending on the result of some expression. Use an if block for that! 
var z = true
if (z) {
    console.log('z is true!')
}
Now change the value of z: 
var z = false
if (z) {
    console.log('z is true!')
}
...the message no longer gets printed! There is another type of block that can follow an if block that will fire if the if condition fails: else! 
var z = false
if (z) {
    console.log('z is true!')
}
else {
    console.log('z is false!')
}
You can put any single expression into an if's condition. The value of the expression will be coerced into a boolean the same as wrapping a call to Boolean() around the value yourself.
You can create boolean expressions to create lots of types of tests. Here are some operators that perform tests and return a boolean value:
operator name
< less than
<= less than or equal to
> greater than
>= greater than or equal to
== is equal (coercive)
!= not equal (coercive)
=== is equal (strict)
!== not equal (strict)
Try modifying this program for all of the different types of boolean test operators and different values for n: 
var n = 3

if (n < 5) {
    console.log('n is less than 5!')
}
You could also assign the result of the test to a variable: 
var n = 3
var b = n < 5

if (b) {
    console.log('n is less than 5!')
}
These binary boolean operators will help to string boolean expressions together:
operator name
&& and
|| or
Try building some compound statements like these: 
var x = 3
var y = 2

if (x < 5 && y > 1) {
    console.log('true!')
}
else {
    console.log('false!')
}
You can also chain together more conditional statements with else if: 
var n = 4
if (x <= 3) {
  console.log('x is less than or equal to 3!')
}
else if (x === 8) {
  console.log('x is eight!')
}
else {
  console.log('x is something else!')
}
You can put in as many else ifs as you like!
There is another type of boolean operator which is unary like ++ that we saw previously. It's the NOT operator and it's just a !. It returns the opposite boolean value. 
var b = false
if (!b) {
    console.log('not b is true!')
}
