It's easy to get started using javascript as a calculator.
You can type arithmetic mathematical expressions like 2 + 3 * 5. This is known as infix notation. Try some expressions for yourself in the box below!
Here are some operators to get you started:
symbol operation
+ addition
- subtraction
/ division
* multiplication
% modulo
Note that in the expression, 2 + 3 * 5 the result is 17 instead of 25. As you may remember from algebra class, multiplication (*) is higher up on the order of operations than addition (+). In programming languages this order of operations is often called operator precedence.
Read more about operator precedence in javascript.
This type of interactive, calculator-style environment is called a REPL. REPL is short for Read-Eval-Print-Loop. The REPL reads your input, evaluates the input you gave it, prints the result of the evaluation, and then starts all over again reading input.
Sometimes storing a result for later is convenient. Use the var keyword and the equal sign (=) to give a value a name. 
var x = 3 * 2 + 5
Now whenever you use the x name in an expression, the value of x will be used! You can use whatever names for variables you want, so long as they are all one word with no spaces and no punctuation except for _ and $.
Let's compute the area of a circle! 
var pi = 3.14
var r = 5
var area = pi * r * r
If you type these lines onto the REPL, now area will have the value you wanted. Type area by itself on the REPL to see its value.
We can put all those declarations on the same line by using a semicolon to separate the statements: 
var pi = 3.14; var r = 5; var area = pi * r * r
Javascript has some math routines built into it too! Try these lines in the REPL: 
Math.cos(0)
Math.tan(1 / Math.PI)
Math.sin(Math.acos(0))
These math routines are function calls. More on those later.
