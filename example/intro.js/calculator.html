It's easy to get started using javascript as a calculator.
You can type arithmetic mathematical expressions like 2 + 3 * 5. This is known as infix notation. Try some expressions for yourself in the box below!
Here are some operators to get you started:
symbol operation
+ addition
- subtraction
/ division
* multiplication
% modulo
Note that in the expression, 2 + 3 * 5 the result is 17 instead of 25. As you may remember from algebra class, multiplication (*) is higher up on the order of operations than addition (+). In programming languages this order of operations is often called operator precedence.
Read more about operator precedence in javascript.
This type of interactive, calculator-style environment is called a REPL. REPL is short for Read-Eval-Print-Loop. The REPL reads your input, evaluates the input you gave it, prints the result of the evaluation, and then starts all over again reading input.
Sometimes storing a result for later is convenient. Use the var keyword for that. 
var x = 3 * 2 + 5
Now whenever you use the x name in an expression, the value of x will be used!
