In computer programming, pieces of data have types. For instance, when you use javascript as a typesulator, you're mostly dealing with the Number type.
Here are some builtin types:
  • Number - numbers
  • String - blobs of text
  • Boolean - true or false
  • undefined - placeholder type for empty values
  • Function - a chunk of code
  • Object - a lookup table
  • Array - an ordered list
In javascript you can use the typeof keyword to check what type an object is. Try these lines in the REPL: 
typeof 42
typeof "wooo"
typeof false
Use the REPL to determine the types of the following expressions:
expression type status
55.5 ---
"ahoy thar!" ---
Math.sin(Math.PI) ---
{ a : 2, b : 3, c : 5 } ---
'beep' + 33 ---
5 > 10 ---
true || false ---
There are some built-in functions to help with converting between types. For any of the type functions: Boolean, String, and Number, you can pass an object of a different type to coerce the value into the function's namesake. Try these function calls: 
Boolean('false')
String(5)
Number('22')
Number(true)
