Arrays are similar to objects except that the elements are ordered and javascript provides many built-in functions for manipulating the elements of the array.
Arrays can be created using array literal syntax: 
var xs = [ 3, 4, 5 ]
To see how many elements an array has, use .length: 
var values = [ 'a', 'b', 'c', 'd' ]
console.log(values.length)
Arrays can also be modified. When you .push(x), x gets put onto the end of the array. When you .pop(), the element at the end of the array is removed and its value is returned. 
var xs = [ 9, 8 ]
xs.push(11)
console.dir(xs)
var x = xs.pop()
console.log('x = ' + x)
console.dir(xs)
To see what methods array has built into it, you can do: 
Object.getOwnPropertyNames(Array.prototype)
